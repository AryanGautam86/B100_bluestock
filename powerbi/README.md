# Bluestock Analytics Dashboard

## Project Overview

This project analyzes financial and operational performance of 100+ Indian listed companies using PostgreSQL and Power BI.

The dashboard provides:

* Executive overview of company performance
* Sector-wise analysis
* Revenue and profitability insights
* Company-level deep dive
* Financial KPI tracking
* Interactive filters and slicers

## Tech Stack

* PostgreSQL
* Python
* Pandas
* Power BI
* VS Code

## Dashboard 1: Executive Overview

Features:

* Total Companies
* Total Revenue
* Total Sectors
* Sector-wise Revenue Analysis
* Year Filters
* Sector Filters

## Dashboard 2: Company Deep Dive

Features:

* Company Selector
* Company Profile
* Company Logo
* Revenue Analysis
* ROE Analysis
* Book Value
* Face Value

## Key Findings

* Energy sector generated the highest revenue.
* Banking sector ranked second in revenue contribution.
* Reliance Industries was the highest revenue-generating company.
* Revenue grew significantly from 2013–2024.
* TCS and Infosys demonstrated strong profitability and financial health.
